## Patient Intake and Clinical Analytics Dashboard

This Streamlit-based BioMed app allows healthcare professionals to manage patient intake data and perform basic clinical analytics. 
Features include;

- Patient Intake Form: Collect demographics, clinical information (blood type, smoking history), and upload lab results via CSV.

- Patient Database Management: Save patient entries to a CSV database for future reference.
